# Divina Cocina  
1. Clona el repositorio:  
   `git clone [tu-enlace-de-github]`  
2. Configura la base de datos:  
   - Importa `database.sql` en phpMyAdmin.  
3. Inicia el backend:  
   - Coloca los archivos PHP en `htdocs` (XAMPP/MAMP).  
4. Inicia el frontend:  
   - `npm install`  
   - `npm run dev`
